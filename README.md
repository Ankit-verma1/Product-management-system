# Product-management-system
A Java-based Product Management System developed using JSP, Servlets, JDBC, and Core Java, implementing OOP concepts and CRUD operations


## Technologies Used
- Core Java
- JSP
- Servlet
- JDBC
- MySQL
- HTML/CSS
- Apache Tomcat

## Features
- Add new products
- View product list
- Update product details
- Delete products
- Search products by ID

- ## Project Structure
- Controller: Servlets
- View: JSP pages
- Model: Java classes
- Database: MySQL

## How to Run
1. Clone the repository
2. Import project into Eclipse/IntelliJ
3. Configure MySQL database
4. Deploy on Apache Tomcat
5. Access via browser
